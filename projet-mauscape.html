Mauscape

2023 · Premier projet Java

Jeu d'aventure textuel de type "escape game" jouable dans le terminal. Le joueur explore un château mystérieux, résout des énigmes et collecte des objets pour s'échapper. Mon premier projet complet de jeu qui a renforcé ma passion pour le développement logiciel.

Architecture du Code

Le jeu est structuré autour de classes représentant les différents éléments : les pièces, les objets, le joueur et le moteur de jeu principal qui gère les commandes et l'état de la partie.

Structure des classes

src/ ├── Main.java ├── Jeu.java // Moteur principal ├── Joueur.java // État du joueur et inventaire ├── Piece.java // Représentation d'une salle ├── Objet.java // Items collectables ├── Commande.java // Parsing des commandes └── Sauvegarde.java // Système de save/load

Exemple de gestion des commandes

public void traiterCommande(String input) { String[] parts = input.toLowerCase().split(" ", 2); String action = parts[0]; String cible = parts.length > 1 ? parts[1] : ""; switch (action) { case "aller": deplacer(cible); break; case "prendre": prendreObjet(cible); break; case "examiner": examiner(cible); break; case "utiliser": utiliserObjet(cible); break; default: System.out.println("Commande non reconnue."); } }

Défis et Solutions

Scénarios à embranchements : Gérer les différents chemins possibles dans l'histoire a nécessité une structure de données adaptée. J'ai utilisé un graphe de pièces connectées avec des conditions de passage.

Parsing des commandes : Interpréter les commandes du joueur de manière flexible (synonymes, différentes formulations) a été un défi intéressant résolu avec un système de reconnaissance de mots-clés.

Système de sauvegarde : Implémenter la sérialisation de l'état du jeu pour permettre au joueur de reprendre sa partie m'a appris les bases de la persistance des données en Java.

Apprentissages

Ce premier projet de jeu m'a enseigné les fondamentaux de la programmation Java : structures de contrôle, collections, classes et objets. C'est ce projet qui a vraiment déclenché ma passion pour le développement.

J'ai également découvert l'importance du game design : équilibrer la difficulté, créer une atmosphère immersive et guider le joueur subtilement sont des compétences transférables à tout projet d'interface utilisateur.
