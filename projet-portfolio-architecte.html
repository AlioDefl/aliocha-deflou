Exemple de composant

interface ProjectCardProps { title: string; image: string; category: string; } const ProjectCard: React.FC<ProjectCardProps> = ({ title, image, category }) => { return ( <motion.div whileHover={{ y: -10 }} className="project-card" > <img src={image} alt={title} loading="lazy" /> <h3>{title}</h3> <span>{category}</span> </motion.div> ); };

Défis et Solutions

Optimisation des images : Les images haute résolution d'architecture étant lourdes, j'ai mis en place un système de lazy loading et de formats optimisés (WebP) pour garantir des temps de chargement rapides.

Animations performantes : Pour les animations au scroll, j'ai utilisé Framer Motion avec l'Intersection Observer API pour déclencher les animations uniquement quand les éléments sont visibles.

Responsive design : L'affichage des projets en grille nécessitait une adaptation soignée selon les tailles d'écran, résolue avec CSS Grid et des breakpoints Tailwind personnalisés.

Apprentissages

Ce projet m'a permis d'approfondir mes compétences en React et TypeScript, notamment sur le typage strict des props et la création de composants réutilisables.

J'ai également découvert les bonnes pratiques d'optimisation web : Core Web Vitals, lazy loading, et l'importance du choix des formats d'image pour la performance.
