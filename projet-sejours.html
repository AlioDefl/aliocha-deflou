Séjours Linguistiques

2025 · Projet BUT Informatique

Application de gestion de séjours linguistiques avec interface graphique complète. Ce projet académique m'a permis de développer mes compétences en conception d'interfaces utilisateur avec Java Swing et en gestion de données persistantes.

Java Swing IHM JDBC MVC

Fonctionnalités

Gestion des Participants

CRUD complet pour les élèves, familles d'accueil et accompagnateurs

Planification des Séjours

Création et modification de séjours avec destinations et dates

Affectation Automatique

Algorithme d'affectation des élèves aux familles selon critères

Export Documents

Génération de fiches récapitulatives et listes de participants

Architecture Technique

L'application suit le pattern MVC (Model-View-Controller) pour une séparation claire des responsabilités. La couche de persistance utilise JDBC pour la connexion à une base de données SQLite locale.

Structure du projet

src/ ├── model/ │ ├── Eleve.java │ ├── Famille.java │ ├── Sejour.java │ └── Affectation.java ├── view/ │ ├── MainFrame.java │ ├── ElevePanel.java │ ├── SejourPanel.java │ └── AffectationDialog.java ├── controller/ │ ├── EleveController.java │ └── SejourController.java └── dao/ ├── EleveDAO.java └── SejourDAO.java

Exemple d'interface Swing

public class ElevePanel extends JPanel { private JTable table; private EleveTableModel model; public ElevePanel() { setLayout(new BorderLayout()); model = new EleveTableModel(); table = new JTable(model); JScrollPane scrollPane = new JScrollPane(table); add(scrollPane, BorderLayout.CENTER); JPanel buttonPanel = createButtonPanel(); add(buttonPanel, BorderLayout.SOUTH); } }

Défis et Solutions

Interface utilisateur intuitive : Concevoir une interface claire avec Swing a nécessité une réflexion approfondie sur l'ergonomie. J'ai utilisé des onglets, des tables triables et des dialogues modaux pour faciliter la navigation.

Persistance des données : La gestion de la base de données avec JDBC a impliqué la création d'une couche DAO (Data Access Object) pour centraliser les requêtes SQL et faciliter la maintenance.

Algorithme d'affectation : Développer un algorithme qui prend en compte les préférences, les langues parlées et la capacité des familles pour proposer des affectations optimales.

Apprentissages

Ce projet m'a permis de maîtriser le développement d'interfaces graphiques avec Java Swing, de la création de composants personnalisés à la gestion des événements utilisateur.

J'ai également approfondi mes connaissances en SQL et en architecture logicielle, notamment l'importance du pattern MVC pour organiser le code de manière maintenable.
